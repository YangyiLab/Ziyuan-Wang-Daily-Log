## 机器学习基础

### 数据结构

N-维数据

+ 0维数组 浮点数
+ 1-d [1.0,2.7,3.4] 一个样本
+ 2-d 特征矩阵 多个样本
+ 3-d RGB图片 宽、高、通道
+ 4-d 一个batch的图片

[1:3,:] 1 2 行 [1,3)

dim = 0 按行合并 向下堆 [3,4] [3,4] [6,4]
dim = 1 按列合并 向又加

### 预处理

用 readcsv 读

处理缺失 插值法
```py
inputs = inputs.fillna(inputs.mean())
```

### 优化算法

+ 小批量梯度下降 随机梯度下降 比 梯度下降慢 加大批量时随机梯度下降比较快
+ 冲量法 使用平滑过的梯度进行更新 
+ Adam 做了很多的平滑 对学习率不敏感
